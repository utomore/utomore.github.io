VSCode-vcpkg-CMake

In my free time, I tried to build the enviroment for my side-project which I use to learn opengl.

I chose vscode + vcpkg(windows) + cmake as my coding environment.

Of course using visual studio is the easier way to build with vcpkg and I have done it before. So today I chose a different approach to build the side-project.

Installation

First of all, you need to install the software: vcpkg, CMake, VScode, including the following extensions: C/C++, CMake, CMake Tools.

Depending on your compiler, you may need to install Visual Studio or MinGW(for GCC, Clang).

Setting Up vcpkg

To install vcpkg, clone the vcpkg git repository from github and run the bootstrap-vcpkg.bat script.

You can add the path of vcpkg folder to the Path environment variable.

  • Type “env” in your Start menu, and then add the path to the Path variable.
