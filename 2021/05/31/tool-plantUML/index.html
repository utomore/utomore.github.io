PlantUML 介紹

UML 介紹

當需要視覺化物件間的關係 UML 時，可以直接快速的使用對應的程式格式顯示出想要的結果，雖然可能圖案不是非常美觀，但主要適用於快速規劃出一個架構，讓團隊中每個人的認知是統一的。

有各式各樣的 UML 繪製工具，其實使用 ppt 或是 draw.io 也可以畫，工具並沒有高低之分，主要是看自己適合哪一種。這邊將介紹 plantUML 結合 vscode 來進行繪製。

PlantUML 支援種類

PlantUML是一個開源的 UML 繪製工具，主要提供以下種類圖的繪製:

  • Sequence diagram
  • Usecase diagram
  • Class diagram
  • Activity diagram
  • Component diagram
  • Deployment diagram
  • State diagram
  • Timing diagram

PlantUML 環境安裝

Dependency: VScode + PlantUML plugin + Java SE Development Kit (+ Graphviz)

官網寫沒有安裝 Graphviz 就只能繪製 Sequence Diagram，但是我都可以繪製，所以暫時沒裝。

首先必須先下載 VScode : https://code.visualstudio.com/
下載完後到延伸模組搜尋: PlantUML，如下圖:

套件安裝完之後可以隨意開啟一個資料夾，嘗試上方的程式碼。利用快捷鍵Alt+d，可以在右方直接顯示正在輸入的結果圖形，同步調整。

如果這時候顯示紅色驚嘆號，表示需要下載 Java 開發套件
Java SE Development kit 下載: https://www.oracle.com/java/technologies/javase-jdk16-downloads.html

安裝完後，重新啟動 vscode 就可以看到圖形了。

匯出圖片

使用 vscode 快捷鍵 ctrl + shift + p 輸入 plantuml，可以看到一個匯出目前圖表，點選後選擇輸出的格式即可。
