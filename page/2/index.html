SQLite(2) Command

SQLite學習- 安裝，命令，語法

1. 安裝

選擇適合自己系統的下載安裝
http://www.sqlite.org/download.html

2. 命令

與關係型資料庫進行交互的標準SQLite命令類似於SQL，這些命令基於他們的操作性質可分為以下幾種 :

2.2 DDL-數據定義語言

  • CREATE: 創建一個新的表。一個表的視圖，或是資料庫中其它的對象
  • ALTER: 修改資料庫中的某個已經有的資料庫對象，比如一個表。
  • DROP: 刪除整的表單，或是表的視圖。或資料庫中其它的對象。

2.3 DML-數據操作語言

  • INSERT: 創建一條紀錄
  • UPDATE: 修改紀錄
  • DELETE : 刪除紀錄

  • 2.4 DQL-數據查詢語言

  • SELECT: 從一個或多個表中檢索某些紀錄

3. 語法

SQLite 是不區分大小寫的。但也有些命令是大小寫敏感的。

SQLite(1) 介紹

SQL與MySQL差異

兩者大致功能差異

  • SQLite: 講究功能簡約、小型化、追求最大磁碟效率
  • MySQL: 功能全面、綜合化、追求最大並行處理效率

使用時機

  • SQLite: 單機上使用，需要方便多方移植，頻繁讀寫
  • MySQL: 滿足多用戶同時訪問，訪問量較大
